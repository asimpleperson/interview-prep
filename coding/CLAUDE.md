# coding/

## Purpose

Algorithm and data structure practice targeting staff-level coding rounds. Covers all major LeetCode patterns with a focus on pattern recognition, clean implementation, and optimal complexity.

## Contents

- `practice-checklist.md` — Master checklist organized by topic (arrays, trees, graphs, DP, etc.). Check off problems as completed.

## Conventions

- Track completion status in `practice-checklist.md` using `[x]` checkboxes.
- Add new problems under the relevant topic section with a LeetCode link.
- When a pattern is fully internalized, note the key insight in the **Key Concepts** block for that section.
- Aim to solve each problem within 20–30 minutes under interview conditions.

## Priority Order

1. Arrays, Strings, Two Pointers, Sliding Window
2. Trees, Graphs, BFS/DFS
3. Dynamic Programming
4. Heaps, Tries, Union-Find
5. Greedy, Backtracking, Bit Manipulation

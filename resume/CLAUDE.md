# resume/

## Purpose

Resume source files and exports. Keep this up to date — interviewers will reference it during project deep dive and behavioral rounds.

## Contents

- `Resume.txt` — Plain text source for easy editing and diffing
- `ZhihaoHuResume20260415Final.pdf` — Latest exported PDF (dated 2026-04-15)

## Conventions

- Name PDF exports with the date: `ZhihaoHuResume<YYYYMMDD>.pdf`
- Keep `Resume.txt` as the canonical editable version.
- When tailoring for a specific company, create `Resume-<Company>.txt` and export a matching PDF.

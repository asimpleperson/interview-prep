# Interview Prep

Preparation materials for staff-level software engineer interviews at top tech companies including Google, Meta, ByteDance, Databricks, Uber, Amazon, and Apple — as well as AI-focused companies like OpenAI and Anthropic.

## Goal

Land a staff+ software engineer offer by systematically preparing across all four interview dimensions.

## Repo Structure

```
coding/            Algorithm and data structure practice (LeetCode patterns)
system-design/     Distributed systems design notes and write-ups
behavioral/        STAR stories and leadership principle responses
project-deep-dive/ Deep technical walkthroughs of past projects
companies/         Company-specific prep (placeholder folders per company)
resume/            Resume source text and PDF exports
```

## Interview Sections

### 1. Coding
Algorithm and data structure problem solving. Focus on pattern recognition, optimal time/space complexity, and clean implementation under pressure. See [`coding/`](coding/).

### 2. System Design
Large-scale distributed systems design. Cover scalability, reliability, consistency tradeoffs, and real-world architectural decisions. See [`system-design/`](system-design/).

### 3. Behavioral Questions
Leadership, cross-functional collaboration, conflict resolution, and impact-driven storytelling using the STAR method. Tailored to staff-level scope and influence. See [`behavioral/`](behavioral/).

### 4. Project Deep Dive
In-depth technical walkthrough of past projects — architecture decisions, tradeoffs made, challenges overcome, and measurable outcomes. See [`project-deep-dive/`](project-deep-dive/).

## Company-Specific Prep

Individual folders under [`companies/`](companies/) for Google, Meta, ByteDance, Databricks, Uber, Amazon, Apple, OpenAI, and Anthropic. Populated when interviews are scheduled.

# Interview Prep — Repo Guide

## Purpose

This repo contains all preparation materials for staff-level software engineer interviews at:
- **Big Tech:** Google, Meta, ByteDance, Databricks, Uber, Amazon, Apple
- **AI Companies:** OpenAI, Anthropic

The goal is a staff+ offer. Preparation is organized into four interview dimensions plus company-specific research.

## Folder Structure

```
coding/            Algorithm and data structure practice
system-design/     Distributed systems design study and notes
behavioral/        STAR-method stories and leadership principles
project-deep-dive/ Deep technical walkthroughs of past projects
companies/         Company-specific prep (culture, values, past Qs)
resume/            Resume source and PDF exports
```

## How to Use This Repo

- Each folder has its own `CLAUDE.md` describing its contents and conventions.
- Work through `coding/` daily; treat it as a checklist.
- `system-design/` and `behavioral/` are best reviewed in mock-interview sessions.
- `project-deep-dive/` should be updated whenever a new project narrative is refined.
- `companies/` is populated per-company as interviews get scheduled.

## What They Do

Snowflake is a cloud data platform ("Data Cloud") — a fully managed data warehouse / lakehouse
built on a separation of storage and compute. Engineering spans distributed query execution,
storage, metadata services, and large-scale multi-tenant infrastructure.

## Domain Topics to Know

- Distributed / vectorized query execution and optimization
- Separation of storage and compute (virtual warehouses)
- Micro-partitions, pruning, columnar storage on object stores
- Metadata/catalog services, concurrency, ACID on cloud storage
- Multi-tenant isolation and resource governance

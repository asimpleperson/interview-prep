## Prep Checklist for This Role

### Coding
ByteDance / TikTok is known for difficult algorithm problems. Expect hard LeetCode; first round is often a timed coding screen.
- [ ] Dynamic programming — all Must-Have + High-Priority
- [ ] Graph algorithms (BFS, DFS, Dijkstra, topological sort)
- [ ] Monotonic stack
- [ ] Heap / priority queue
- [ ] Sliding window and two pointers
- [ ] Backtracking
- [ ] Binary search

### System Design
Monetization Technology system design problems center on ads infrastructure.
- [ ] Ad auction system (second-price auction, pacing, budget management)
- [ ] Real-time bidding pipeline (RTB, DSP/SSP, sub-100ms latency)
- [ ] Click / impression tracking at massive scale (Kafka, Flink, ClickHouse)
- [ ] Ad targeting: user segmentation, lookalike, contextual signals
- [ ] Attribution and measurement (multi-touch, conversion tracking)
- [ ] Fraud detection in ad networks

### Behavioral / Tech Lead

- [ ] "Describe a time you made a high-stakes architectural decision under time pressure"
- [ ] "How do you handle underperforming teammates?"
